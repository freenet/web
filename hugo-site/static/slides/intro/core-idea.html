The Core Idea

  • Freenet is a global, decentralized, scalable key-value store
  • The twist: keys are WebAssembly code, not just addresses
  • Contract code specifies what values are valid, how to merge states, how to sync
  • key = hash(hash(contract_code) + contract_parameters)
  • State can be anything: chat, documents, social graphs â€” just bytes
  • Three building blocks: Contract (public shared state), Delegate (private user state), UI (web app in browser)
  • Delegate = user's agent, handles keys/signing/encryption locally
