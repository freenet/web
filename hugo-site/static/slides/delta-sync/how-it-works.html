How Delta Sync Works

  1. Summarize - Generate compact hash of current state
  2. Exchange - Peers swap summaries
  3. Diff - Calculate what the other peer is missing
  4. Send delta - Transmit only the differences
  5. Merge - Apply delta to local state

Result: Bandwidth proportional to changes, not total state size

The key insight is that summaries are small fixed-size representations that let peers identify what they're missing without downloading everything.
