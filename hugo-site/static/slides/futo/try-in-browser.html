Participation, Not Installation

Before

Hear about Freenet
→ install a node
→ configure it
→ find an app
→ find people

Now

Open freenet.org/try
→ River opens in the browser
→ you're in a live room
no install, no account, no download

This is a hosted demo. It runs on our server, so your data lives with us — encrypted with your access key, so even we can't read it. It's an on-ramp, not the final trust model. Install your own peer anytime and export your data to it in one click.
  • The activation barrier used to be the whole problem: you had to install and configure before you could see anything.
  • Now the first experience can be participation, not installation. Open the page, River loads, you're talking in the Freenet Official room.
  • Be honest and lead with the caveat yourself: the hosted demo is centralized, which is the opposite of what Freenet is for. It exists to lower the barrier.
  • Your data is encrypted with your access key even in the hosted demo, and there's a one-click export to a real peer you run yourself.
  • Line to land: "The first experience no longer has to be installation. It can be participation."
