BACKUP (Q&A / accuracy, not for live):

  • Served-code trust: the hosted /try runs JS we serve, so a compelled operator could ship key-stealing code. That's true of all in-browser crypto. The real answer is running your own peer, which is why export is one click.
  • If the room gets spammed live: that's the point, no one can delete it centrally. Moderation is room-level bans by the owner. Don't fight it on stage.
