freenet-git: Git Without GitHub

Git remains Git. What changes is where the repository lives. 

# the repo is a Freenet contract, not a server you log into
git clone freenet://<repo-key>
git fetch
git push
  • Same workflow developers already know
  • No account, no forge company, nothing to take the repo down
  • Working today: clone, fetch, push
  • This is the hacker-friendly demo, and this audience will like it.
  • The pitch is deliberately boring: Git stays exactly the same. The only thing that changes is that the repository lives on Freenet instead of on a company's servers.
  • Clone, fetch, push all work today. If the live demo is reliable, show a clone. If not, a short terminal recording.
  • The line to land: "Git remains Git. What changes is where the repository lives."
