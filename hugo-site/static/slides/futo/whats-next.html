Not launch-ready — the point is that the same substrate is already taking on this many application shapes

  • Roadmap slide. Be candid about maturity, in decreasing order of readiness.
  • Mail is pre-alpha: it's deployed and you can try it, the crypto is post-quantum, but the repo says don't store anything you'd mind losing. It matters because it takes Freenet from chat to email-shaped messaging.
  • Raven is a prototype -- a social feed. It's valuable as a stress test: social state is high-churn, which is exactly where a network like this gets exercised.
  • Harvest is early design. Say "a direction," not "nearly ready." Don't turn it into a trust-and-safety tangent for a marketplace that isn't built yet.
  • Land the closing line: these aren't all ready, and that's fine. The signal is that one substrate is supporting this many shapes at once.
