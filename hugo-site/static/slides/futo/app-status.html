green = use it today  •  yellow = try it, it'll break  •  grey = design, not software yet

  • This is the honest version of the ecosystem map. Green means you can use it today. Yellow means you can try it but it will break. Grey means it's design, not software yet.
  • River is the anchor -- the one I'll demo, and private rooms are the new capability since we last spoke.
  • Delta and freenet-git are genuinely running on the network. Atlas has a first working version running -- not "we solved decentralized search."
  • Mail is pre-alpha: deployed and post-quantum, but the repo literally says don't store anything you'd mind losing. Say that out loud.
  • Harvest is early design -- a direction, don't oversell it. Raven is a promising prototype.
  • The point isn't that all seven are shipped. It's that one substrate now supports this many application shapes.
