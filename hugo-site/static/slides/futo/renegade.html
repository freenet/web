Blended conservatively with the existing router • live in Freenet since March 2026 • renegade-ml

SAY (cues):

  • Routing isn't only heuristic now. The network learns from how peers actually behaved.
  • Greedy rule: forward to the closest peer. But closest isn't always best (slow, overloaded, dropping requests).
  • Renegade learns four things per routing event: peer, contract, ring distance, time.
  • From that it predicts: will this peer succeed, and how fast?
  • Clever part is peer times contract: catches a peer censoring one contract while looking fine on everything else.
  • Blended conservatively with the existing router, not trusted blindly.
  • Don't go deeper into the ML unless asked. The point: serious engineering, not AI sprinkled on top.
