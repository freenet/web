What's New: Private Rooms

Encrypted shared application state, hosted by a network that can't read it

Room secret

AES-256-GCM encrypts every message

Key distribution

ECIES (X25519) gives the secret to members only

Rotation

Weekly, and on every ban

Peers host and sync the room and enforce its rules, without ever reading what's inside.

SAY (cues):

  • Shared encrypted app state, on a network nobody operates. The new thing since last talk.
  • Every message encrypted: per-room AES-256-GCM secret.
  • Members get the secret via ECIES (X25519). Only members decrypt.
  • Rotates weekly + on every ban. Removed member can't read anything new.
  • Peers enforce rules + sync updates, never read content.
  • Room is an application object, not a row in a database.
