Atlas: Discovery Without a Platform

If apps and content don't live on platforms, discovery can't depend on a platform either.

Atlas is a decentralized directory for Freenet — a way to find apps, sites, and links with no central search engine in the middle.

A first version is running on Freenet today

  • Discovery is the question everyone asks about decentralized systems: if there's no platform, how does anyone find anything?
  • Atlas is our answer -- a directory and search layer that is itself just a Freenet app, not a service we run in the usual sense.
  • Set expectations honestly: a first version is live. It's real, and it's early. Over the next few slides I'll show exactly what's decentralized today and what isn't.
  • The reason Atlas is a good example is that its architecture draws one line very deliberately -- and that line is what lets it grow.
