SAY (cues):

  • Discovery = the quiet power center of the web.
  • Whoever runs search / the store / the feed decides what billions find. For advertisers and engagement, not you.
  • Atlas: discovery hosted across the network, no company owning that position. Transparent, not a black box.
  • The real point: you choose which index you trust and which algorithm ranks it. We ship good defaults, you're never locked in.
  • How it's built: a crawler indexes content, and a separate step writes neutral descriptions, today with an LLM. None of it is fixed, swap any part.
  • On the right: live Atlas UI on Freenet. Real and early.
  • No-gatekeeper is the goal. Today the default index and ranking are still run by us.

BACKUP (Q&A / accuracy, not for live):

  • Scale: don't say "anyone hosts the whole index", it won't stay small. It's hosted decentralized on the network; users pick which index(es) they trust.
  • No-gatekeeper is the GOAL, not today's reality. The default index and ranking are still run by us, with good defaults.
