Centralized today

One crawler, one describer — we run them. We decide what gets indexed.

Already decentralized

Storage, serving, and integrity — any node hosts, serves, and verifies the index.

  • Here's the honest pipeline. A crawler we run fetches pages. An LLM writes a short title, summary, and tags for each one. That becomes a signed entry, which goes into an index contract that lives on Freenet. Your browser pulls the index and searches it locally.
  • Say the quiet part out loud: the crawler and the describing LLM, we run those. We currently decide what gets indexed. That's the centralized part, and I'm not going to hide it.
  • What's genuinely decentralized already: the index itself. Any node can host it, serve it, and verify it -- it's signed, so nobody has to trust the server they got it from.
