Combine Them Like LEGO

Every app's state is an open contract on the same network. Any app can build on any other, no silos, no permission. Interoperability is the default.

freenet-git + River → Discuss code right where it lives
Delta + River → A published post with a real comment thread
Ghost Keys + Harvest → Reputation that's yours, across every app

SAY (cues):

  • The multiplier: on Freenet, apps interoperate by default. No silos.
  • Every app's state is an open contract on the shared network, so any app can build on any other's data. No API, no permission, no business deal.
  • Combine them like LEGO. Git plus River: discuss code in context. Delta plus River: a blog with real comments. Ghost Keys plus Harvest: portable reputation.
  • On the web this needs OAuth, APIs, deals, and the platform can cut you off. Here it's free.
  • Land: interoperability isn't a feature we added. It's what you get when there are no platforms.
